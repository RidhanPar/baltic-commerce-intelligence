# CV.lv Requirement Analysis

Research date: June 7, 2026

## Bottom Line

The strongest portfolio for the Latvian data analyst market should lead with advanced SQL, Power BI/DAX, Excel, data modeling, data quality, business problem solving, and stakeholder communication. Python or R, experimentation, ETL/ELT, cloud warehouses, dbt, Git, and product or marketing analytics materially strengthen a candidate.

Snowflake and Looker appear in senior modern-stack roles. Databricks is valuable platform breadth, but it is less consistently requested in the reviewed CV.lv analyst listings than Power BI, SQL, Excel, and general warehouse skills.

## Requirement Priority

### Tier 1: Must Be Obvious in the Portfolio

- Advanced SQL: joins, CTEs, aggregations, windows, optimization
- Power BI: semantic modeling, dashboards, DAX, performance
- Excel: formulas, pivots, Power Query, variance and planning analysis
- Data modeling and warehouse concepts
- Data quality, reconciliation, and trusted metrics
- Business acumen and actionable recommendations
- Clear communication for non-technical stakeholders

### Tier 2: Strong Differentiators

- Python or R for analysis and automation
- A/B testing, statistics, cohorts, funnels, segmentation, LTV, CAC, ROI
- ETL/ELT and automated reporting
- dbt and Git-based workflow
- Product, marketing, finance, or logistics domain analysis
- Requirements gathering and metric documentation

### Tier 3: Modern-Stack Breadth

- Snowflake or similar cloud warehouse
- Looker semantic modeling
- Databricks or Spark for high-volume event processing
- Data governance, metadata, privacy, and row-level access
- Event tracking tools and self-service analytics
